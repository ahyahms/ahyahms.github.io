Tentang Game

Aliens Math adalah permainan edukasi berhitung

Aliens Math Dirancang untuk memenuhi syarat menyelesaikan mata kuliah Animasi Multimedia

Aliens Math adalah game edukasi yang membantu siswa belajar penjumlahan, pengurangan & perkalian yang menyenangkan!.

  • Game ini dirancang menggunakan apliaksi Construct 3.
  • Bertemakan game edukasi referensi dari Saksham Nanda developer asal India yang merancang game ini secara open source.
  • Dirancang khusus untuk melatih kecepatan dan ketepatan berhitung anak pada pelajaran Matematika.

Dalam kegiatan ini anak-anak akan belajar matematika dengan menyenangkan. Menyerang pesawat ruang angkasa dengan pertanyaan bergerak turun dari atas layar menuju meriam di bagian bawah. Tembak satu hadiah yang benar dan akan naik level berikutnya.

Tentang Website

Webiste ini dirancang menggunakan framework Bootstrap 4

Webiste ini dirancang untuk memenuhi syarat menyelesaikan mata kuliah Animasi Multimedia

Bootstrap adalah framework web development berbasis HTML, CSS, dan JavaScript yang dirancang untuk mempercepat proses pengembangan web responsive. Alasan menggunkan framework tersebut adalah.

  • Mudah digunakan dan sangat populer di kalangan developer dan desainer web karena struktur filenya yang sederhana.
  • Kompatibel dengan banyak browser, bootstrap memenuhi persyaratan ini melalui kompatibilitasnya dengan versi terbaru web browser utama.
  • Bootstrap Image System, Bootstrap menangani tampilan gambar dan responsivitas menggunakan aturan HTML dan CSS yang telah ditentukan.

Selain bisa digunakan untuk mengembangkan website dengan lebih cepat, Bootstrap adalah framework gratis yang bersifat open-source. Skrip dan syntax yang disediakan Bootstrap bisa diterapkan untuk berbagai komponen dalam desain web.
